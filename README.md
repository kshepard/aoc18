# aoc18

Advent of Code 2018 solutions

### Requirements

 * [stack](https://docs.haskellstack.org/en/stable/README/)
 * [brittany](https://github.com/lspitzner/brittany)
 * [hlint](http://hackage.haskell.org/package/hlint)

### Installation

```make install```

### Format code

```make format```

### Lint code

```make lint```

### Running solutions

```aoc18-dayXX```
